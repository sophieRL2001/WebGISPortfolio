Project 1: DC Homicide Incidents Map

This interactive map displays homicide incidents in DC for 2024. You can click on incidents to see more details and use the filters to explore the data by method or ward.

The map utilizes Mapbox GL JS for rendering and interactivity, visualizing data sourced from public crime incident reports.

Improvements from the first mapbox: - Added a legend for the color bar that appears when you click an icon - Added a filter that users can select, allowing them to see homicides by the month of 2024, by the Ward in DC, or both. - Cleaned up the popup to make it look nicer - Added a hyperlink at the bottom of each popup that searches on Google for the crime and narrows down the search to articles posted on https://mpdc.dc.gov/, the website for the DC police.
